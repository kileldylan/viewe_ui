# Viewe UI 

This repository contains sample `LVGL Pro` project for the `Viewe 2.8” (240×320) ESP32 display`.


## Getting Started

1. `Fork this repository` to your GitHub account.  
2. `Open the project in LVGL Pro` (download LVGL Pro Editor here: [https://pro.lvgl.io/#download](https://pro.lvgl.io/#download)).  
3. Explore the sample screen and create your own UI designs by adding new screens.

## Firmware Reference

For the hardware and firmware required to run these UIs on the Viewe 2.8” board, check the firmware repository:  
[https://github.com/fbiego/lv-viewe-2.8](https://github.com/fbiego/lv-viewe-2.8)
